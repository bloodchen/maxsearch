Web beacons may be invisible to you, but any electronic image or other web programming code inserted into a web page act as a web beacon. Web beacons or similar technologies may be used for a number of purposes, including, without limitation, to count users interacting with the Product’s features or Offers, to monitor how users navigate through various related websites, or to count how many particular articles or links were actually viewed
